# Dotfiles

## Bootstrap

```sh
chezmoi init --apply erikenz
```

```sh
chezmoi init --apply https://github.com/erikenz/dotfiles.git
```

## Chezmoi commands

[chezmoi.io/quick-start](https://www.chezmoi.io/quick-start/)

### Check changes

```sh
chezmoi diff
```

### Apply changes

```sh
chezmoi apply -v
```

### Open chezmoi repository

```sh
chezmoi cd
```
